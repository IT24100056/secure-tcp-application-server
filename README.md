# Secure TCP Application Server

## Project Overview
This project was developed as part of the Network Programming module. It implements a secure multiprocessor TCP application server with custom protocol and session management.

## Features
- TCP client-server communication
- Custom protocol design
- Session management
- Multi-client handling
- Secure communication mechanisms

## Technologies Used

- C
- Ubuntu Linux
- Socket Programming
- TCP/IP Networking
